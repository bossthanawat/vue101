# Single Page Web Application

คือ web application ที่จะโหลด html เพียงตัวเดียว แต่สามารถตอบสนองการใช้งานของผู้ใช้ โดยจะสามารถเรียกใช้ html ตัวอื่นๆ แบบ dynamic ได้. สิ่งที่แตกต่างอย่างชัดเจนระหว่างเวปไซต์ทั่วไปและ SPA คือการลดจำนวนการรีเฟรชหน้าเวป. SPA ใช้ AJAX และ HTML5 ในการติดต่อไปที่ back-end servers เพื่อไปเอาข้อมูลมาแสดงผลที่หน้าเวป โดยที่ไม่ต้องทำการรีเฟรชเวปทั้งหมด ซึ่งกระบวนการในการ render เวปเพจ จะเกิดขึ้นที่ฝั่ง client เกือบทั้งหมด.
